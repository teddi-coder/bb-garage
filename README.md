# BB Garage — Static Site

Static HTML/CSS landing pages for [BB Garage](https://www.bbgarage.net/), Ipswich QLD.

## Pages
- `index.html` — Homepage
- `dyno-tuning.html` — Dyno Tuning
- `patrol-power-pack.html` — Nissan Patrol 3.0 CR Package
- `ranger-power-pack.html` — Ford Ranger 3.2L Package
- `triton-power-pack.html` — Triton / Pajero Sport Package

## Deploy
Static files — deploy via Cloudways by uploading the root directory.
No build step required.
